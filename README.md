Tic-Tac-Toe AI 🎮🤖
A modern and intelligent Tic-Tac-Toe game where you play against a smart AI powered by the Minimax algorithm. The AI plays strategically, making it challenging to win!

🚀 Features
✅ Smart AI Opponent – Uses Minimax for optimal moves.
✅ Modern UI – Sleek design with smooth animations.
✅ Win Highlights – Winning row gets highlighted.
✅ Auto Restart – Game resets automatically after a win or draw.
✅ Responsive Design – Works on both mobile and desktop.
✅ Dark Theme – Aesthetic dark background for better visuals.

🛠️ Tech Stack
HTML, CSS, JavaScript – Core technologies for smooth gameplay.
Minimax Algorithm – AI logic for challenging moves.
CSS Animations – Smooth hover effects and win highlights.
🎯 How to Play?
Click on an empty cell to place your X.
The AI (O) will make its move.
Try to get three in a row before the AI does!
The game auto-restarts on a win or draw.
